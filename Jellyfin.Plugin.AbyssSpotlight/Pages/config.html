Features

Injects the Abyss theme stylesheet into Dashboard → General → Custom CSS on startup.
Shows the cinematic Continue Watching banner at the top of your home screen. Requires the File Transformation plugin by IAmParadox27.
Automatically sets your home sections to: Continue Watching → Next Up → My Media → Recently Added. (Takes effect on next Jellyfin restart.)

Theme Customisation

Used for buttons, active states, and progress bars. Default: 245, 245, 247 (near-white).
Applied globally to cards, dialogs, and containers. Default: 24px.
Background of episode count and other indicator pills. Default: 55, 55, 55.


Spotlight requires the File Transformation plugin.
If not installed, add this repository to your plugin sources:
https://www.iamparadox.dev/jellyfin/plugins/manifest.json
Then install File Transformation and restart Jellyfin.
